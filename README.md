# React Project Template
React Project Template with automated build.
## Installation
1. Install Node.js
2. Install git
3. In the console, type: npm install
## Usage
To build:
1. npm run build
    a. Currently need to run twice as bundle will fail on first attempt

To test:
1. npm run test

To bundle:
1. npm run bundle
    a. Currently need to run twice as bundle will fail on first attempt
